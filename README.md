# 北京大学选课系统

该系统基于sbt构建，在命令行中输入`sbt run`即可运行该系统。在浏览器输入网址[localhost:8090/index]()即可访问该系统。

该系统的测试账号有

* 学生账号：1，密码：1
* 学生账号：2，密码：2
* 教师账号：11，密码：11
* 教务人员账号：111，密码：111
